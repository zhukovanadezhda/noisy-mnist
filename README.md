This project involves classifying images of handwritten digits from the MNIST dataset, with added noise such as rotations, shifts, zooms, and lighting variations. The dataset includes 60,000 images for training and 10,000 images for testing, all in 28Ã—28 pixel grayscale format. The goal is to develop models capable of accurately recognizing noisy digits. We explore two model architectures:

1. **Fully Connected Neural Networks (FCNN)**.
2. **Convolutional Neural Networks (CNN)**.
